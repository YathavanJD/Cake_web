🎂 Wrap It Up – Cake Shop Website

Wrap It Up is a web-based cake shop application designed to allow customers to explore and order a variety of cakes online. The website provides a simple and attractive interface where users can browse cake products, view details, and place orders easily. The system also includes basic management features to help handle products and customer interactions efficiently.

✨ Features

Browse different cake products
View detailed cake descriptions
Easy-to-use and responsive website interface
Product display and management system
Smooth navigation for customers

🛠️ Technologies Used

HTML
CSS
JavaScript
PHP
MySQL

🚀 How to Run

Clone the repository
Import the database into MySQL
Run the project using XAMPP / WAMP server
Open the project in your browser

🎯 Purpose
This project was developed to practice full-stack web development and to build a functional e-commerce-style website for a cake shop.
