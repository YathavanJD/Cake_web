## 📖 About the Project

**Wrap It Up** is a responsive frontend website designed for a cake and pastry shop. The site allows customers to browse the full cake catalog, explore product details, and place orders online — all through a visually appealing, mobile-friendly design.

This project demonstrates skills in responsive web design, UI/UX development, and clean frontend coding practices using pure HTML, CSS, and JavaScript.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎂 Product Gallery | Browse cakes organized by category with images and prices |
| 📱 Mobile-Friendly | Fully responsive layout for all screen sizes |
| 🛒 Order Form | Simple online order placement with customization options |
| 🎨 Attractive UI | Warm, elegant design suited for a bakery brand |
| 🔎 Product Details | View detailed description for each cake product |
| 📍 Contact Section | Shop location, contact number, and inquiry form |
| 🏠 Landing Page | Eye-catching hero section with featured products |

## ⚙️ Installation

No server required — this is a static frontend project.

```bash
# 1. Clone the repository
git clone https://github.com/YathavanJD/Wrap-It-Up.git

# 2. Navigate to the project folder
cd Wrap-It-Up

# 3. Open in browser
# Simply open index.html in any web browser
# OR use Live Server in VS Code for best experience
```

### Using VS Code Live Server
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. The website opens at `http://127.0.0.1:5500`

---

## 🚀 Usage

1. Open `index.html` in your browser
2. Browse the cake gallery on the homepage
3. Click any product to view its details
4. Fill in the order form to place a custom order
5. Visit the Contact section for shop location and inquiries
